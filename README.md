# N-QUEEN-PROBLEM-8-QUEEN-GAME
My project brings this game to life using C# and .NET Framework that allows for the creation of desktop applications with a user-friendly interface for an interactive experience.

INTRODUCTION & PROBLEM:
   INTRODUCTION:
1.     The Queen Game is a classic puzzle where players place queens on a chessboard without them threatening each other.
2.     Our project brings this game to life using C# and .NET Framework that allows for the creation of desktop applications with a user-friendly interface for an interactive experience.
3.    The 8 Queen Game not only challenges players' strategic thinking but also enhances cognitive skills such as problem-solving and logical reasoning through interactive gameplay.
4.    SQL 2022 is used to store data in database

  Problem Statement:

1.     Challenge: Strategically place queens without allowing them to share rows, columns, or diagonals.
2.     Goal: Implement an engaging game with levels, allowing players to progress by smartly positioning queens.

  Paradigms:

1.     Programming Style: Object-Oriented Programming (OOP) is our go-to approach, modeling chess entities like queens and boards as objects.
2.   Graphical interface: The graphical interface is crafted using c# .net, following the user-friendly event-driven paradigm.
3.   Code Structure: We've organized the code into modular structures, promoting easy maintenance and reusability


     ALGORITHM & EXPLANATION:

 Core Algorithm:
1.     The algorithm ensures queens are placed strategically, checking for threats.
2.     Backtracking is employed to efficiently explore potential solutions.
3.     Data Connection Management: The algorithm establishes a connection to the SQL database         using SqlConnection, ensuring that the connection is open before executing queries.
4.   Data Manipulation: It employs SQL commands to insert and retrieve data, using SqlCommand to execute queries that modify or access the database.


 Features:
1.     Level completion checks track progress.
2.     Dynamic updates on the chessboard and a solution viewer enhance the gaming experience.

 User Interaction:
1.     Players progress intelligently through levels by making well-thought-out queen placements.
2.     The graphical interface dynamically adjusts based on user moves and overall game progress.

 Conclusion:
Our project seamlessly blends strategic problem-solving with an intuitive user interface, providing an enjoyable Queen Game experience
